================================================================================
CUSTOMIZATION OPTIONS
================================================================================

Want to change the behavior? Easy to adjust:

AUTO-ROTATE:
Uncomment the code at bottom of <script> section to make
colors auto-rotate when hovering (2-second intervals)

TRANSITION SPEED:
Change "0.4s" to "0.6s" for slower fades
Change "0.4s" to "0.2s" for faster fades

BUTTON STYLE:
Modify .carousel-btn CSS to change size, color, position

INDICATOR POSITION:
Change .color-indicator bottom/left values to reposition
