HOW IT WORKS
================================================================================

1. CUSTOMER VISITS SHOP PAGE
   → Sees all products with their default color

2. HOVERS OVER A PRODUCT
   → Arrow buttons appear
   → Color indicator appears at bottom

3. CLICKS ARROWS
   → Smoothly transitions to next/previous color
   → Shows color name
   → Sees actual mockup image for that color

4. CLICKS "CUSTOMIZE"
   → Goes to full customization page
   → Can select any color and add their design

================================================================================
CUSTOMIZATION OPTIONS
================================================================================

Want to change the behavior? Easy to adjust:

AUTO-ROTATE:
Uncomment the code at bottom of <script> section to make
colors auto-rotate when hovering (2-second intervals)

TRANSITION SPEED:
Change "0.4s" to "0.6s" for slower fades
Change "0.4s" to "0.2s" for faster fades

BUTTON STYLE:
Modify .carousel-btn CSS to change size, color, position

INDICATOR POSITION:
Change .color-indicator bottom/left values to reposition
